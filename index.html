Cómo funciona

Cinco pasos para un dictamen
completo y defendible

📁
Persona Humana Persona Jurídica Checklist automático

Creá el expediente del cliente

Iniciá un caso para persona humana o jurídica. Merlin genera automáticamente el checklist documental según el tipo de sujeto y la normativa UIF/BCRA aplicable. El expediente queda en estado BORRADOR hasta completar la documentación requerida.

  • ✓ Tipos: Persona Humana (PH) y Persona Jurídica (PJ)
  • ✓ Checklist documental según normativa vigente
  • ✓ Asignación automática al analista responsable
  • ✓ Historial de cambios desde el primer momento
🤖
OCR + LLM Confidence Score Clasificación automática

El sistema extrae y estructura la información

Subí los documentos requeridos. Merlin los clasifica por tipo (DNI, constancia AFIP, estatuto, balance, etc.), aplica OCR e IA para extraer los campos clave, y asigna un score de confianza a cada dato extraído.

  • ✓ Extracción automática vía Gemini Vision API
  • ✓ Score de confianza campo por campo (0-100%)
  • ✓ Flags automáticos para campos de baja confianza
  • ✓ Detección de inconsistencias entre documentos
✏️
Revisión humana Corrección manual Historial de cambios

El analista valida o corrige los campos

La interfaz presenta todos los campos extraídos con su score de confianza. Los campos con confianza baja se marcan automáticamente para revisión prioritaria. Cada corrección queda registrada con el usuario que la realizó.

  • ✓ Vista campo por campo con evidencia documental
  • ✓ Corrección inline con justificación
  • ✓ Comparación valor extraído vs valor corregido
  • ✓ Todo queda en el log de auditoría
⚙️
Motor UIF/BCRA Screening AML Score trazable

Evaluación normativa automática end-to-end

El motor de reglas evalúa el perfil contra 40+ reglas UIF/BCRA. El screening AML verifica contra listas PEP, OFAC y Naciones Unidas. El score de riesgo se calcula con cada factor documentado y justificado.

  • ✓ 40+ reglas configurables sin necesidad de deploy
  • ✓ Hallazgos con severidad: Info / Warning / Crítico / Bloqueante
  • ✓ Screening PEP, OFAC y ONU
  • ✓ Score 0-100 con factores positivos y negativos documentados
📋
Dictamen operativo Export PDF Auditoría inmutable

El Oficial emite el dictamen final

El Compliance Officer revisa el dictamen completo: perfil del cliente, hallazgos, score de riesgo y límites operativos sugeridos. Puede aprobar, rechazar, pedir subsanación, o hacer overrides con justificación obligatoria.

  • ✓ Dictamen en pantalla + exportación PDF profesional
  • ✓ Límites operativos con justificación normativa campo a campo
  • ✓ Override de límites con nota obligatoria
  • ✓ Log de auditoría inmutable de todo el proceso

Marco regulatorio

Construido sobre la normativa
argentina vigente

Cada regla del motor está respaldada por normativa específica. Y cuando la regulación cambia, el motor se actualiza sin nuevo deployment.

UIF
Unidad de Información Financiera Res. 30/2017 · 156/2018 · 76/2019 y modificatorias
  • Debida diligencia del cliente (DDC)
  • Identificación y verificación de PEP
  • Matrices de riesgo para PSPs y PSAVs
  • Onboarding y actualización de datos
BCRA
Banco Central de la República Argentina Com. A 6859 · 7429 · 7526
  • Límites operativos para PSPs
  • Prevención de LA/FT en el sistema financiero
  • Marco regulatorio de activos virtuales
  • Requisitos KYC para apertura de cuenta
ARCA
ARCA (ex AFIP) Validación fiscal y categorial
  • Validación de CUIT/CUIL
  • Categorías fiscales: monotributo / RI / empleador
  • Constancias de inscripción online
  • Consistencia de actividad declarada
GAFI
Grupo de Acción Financiera Internacional 40 Recomendaciones · EBR
  • Enfoque Basado en Riesgo (EBR)
  • Rec. 10 — Debida diligencia del cliente
  • Rec. 15 — Nuevas tecnologías y VA
  • Metodología MONEYVAL compatible

Por qué Merlin

No es otro software de compliance.

01
💡

IA Explicable. No caja negra.

Cada decisión puede ser leída, entendida y cuestionada por cualquier auditor externo. La lógica regulatoria es siempre visible, campo por campo. Si la UIF pregunta, tenés respuesta.

02

El humano siempre decide.
